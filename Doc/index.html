Welcome to the Marketing Bundle Documentation

3DEXCITE Marketing Bundle is a web application that allows playing experience.

IMPORTANT:
For optimal performance we strongly recommend to use a browser based on Chromium.

Parts of the bundle code are based on WebGL technology which is not yet available in all the browsers, or only partially supported.

    Consequently, the bundle is not supported by the following browsers:
  • Mozilla Firefox on MacOS
  • Safari on MacOS

Benefits:

  • Easy to integrate, respects web standards
  • Pure experience play (scene, behaviors)
  • Visual excellence and consistency

Highlights:

  • Bundle integration
  • Parameters control
  • Communication with html pages
