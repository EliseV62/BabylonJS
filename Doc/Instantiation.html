Introduction

The codes below show how to integrate the 3DEXCITE Standalone Player Web into an html page with a .dsxm sample file:

Warning: name in Web Component tag is in lowercase

Using html


                    <script type="text/javascript" src="./webGL_HMI_34WebComponent.js"></script>
                    <bundle-3ds-webgl_hmi_34 [option=value]…></bundle-3ds-webgl_hmi_34>

Using javascript


                     let js = document.createElement("script");
                     js.type = "text/javascript";
                     js.src = "./webGL_HMI_34WebComponent.js";
                     document.body.appendChild(js);
                     let bundle = document.createElement("bundle-3ds-webgl_hmi_34");
                     bundle.setAttribute('option', 'value');
                     let playerContainer = document.getElementById("PlayerContainer");
                     playerContainer.appendChild(bundle);

Using Player.js helper


                    <script type="text/javascript" src="./PlayerHelper/Player.js">
                    <script type="text/javascript" src="./webGL_HMI_34WebComponent.js"></script>
                    let player = new Player("webGL_HMI_34Player");
                    let bundle = player.createBundle("bundle-3ds-webgl_hmi_34" [, customSplashscreen]);
                    bundle.setAttribute('option', 'value');
                    let playerContainer = document.getElementById("PlayerContainer");
                    playerContainer.appendChild(bundle);
