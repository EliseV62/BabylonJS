Setup

Prepare

The bundle zip contains all files and resources necessary to play the experience on an HTTP web server.

You simply need to unzip the bundle on a server then run the .html sample.

Local testing

For local testing you can use the web server for Chrome as HTTP web server:

1. Installation https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb?hl=en

2. Launch chrome://apps

3. Configuration
Set the bundle directory and click on the web server URL link:

4. From the web page opened automatically on your browser Click on the .html sample
